https://web.archive.org/web/19990223234656/http://caesar.ics.uci.edu/juice/ [ http://www.ics.uci.edu/~juice ]


SOFTWARE REQUIREMENT:
  
  Prior to installing these source files, download the latest Windows Oberon
  System 3 Release (no earlier than 2.2!) 
  Windows Oberon System 3 is available from:

     ftp.inf.ethz.ch in directory /pub/software/Oberon/System3/Win32/
     ftp.ics.uci.edu in directory /pub/oberon/System3/Windows/Win32/


INSTALLATION

  1) Decode the file "Juice_1.1_Sources_(Intel).zip" with a unzip tool
     able to handle long file names (e.g. WinZIP or WinPack) into 
	 a new directory.
  
  2) For further information please refer to the source documentation in 
     Juice.Build
